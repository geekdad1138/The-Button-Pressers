# The Ghost

## **The Ghost: Satoshi Nakamoto (2008)**

### **The Human**

It’s 2008. The global financial system is melting down. Faith in institutions is at an all-time low. Out of nowhere, a paper appears on a cryptography mailing list signed by "Satoshi Nakamoto." To this day, we don't know if Satoshi was a man, a woman, or a group of people.

### **The Button**

Satoshi "pressed the button" on **Trustless Consensus**. By combining existing ideas like hashcash and digital signatures into a "Blockchain," Satoshi solved a problem that had stumped computer scientists for decades: how to make a digital system work without a central authority. On January 3, 2009, Satoshi ran the code for the "Genesis Block," including a hidden message about bank bailouts in the code itself.

### **The Ripple**

Satoshi stayed for a while, fixed some bugs, and then famously vanished—the ultimate "button press" of walking away.

* **The New Economy:** It triggered the birth of Bitcoin and a trillion-dollar cryptocurrency industry.
* **Decentralized Logic:** Beyond money, it proved that code can be law. It birthed the idea of "Web3" and decentralized AI, where power isn't held by one company, but by the network itself.
